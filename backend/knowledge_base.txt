Trading Strategies

1. EMA Crossover

File: 

backend/strategies/ema_crossover.py

Description: Uses Exponential Moving Averages to identify trend changes.

Logic:

Buy Signal: When 9-period EMA crosses above 21-period EMA

Sell Signal: When 9-period EMA crosses below 21-period EMA

Confidence: Based on EMA separation and volume

Use Case: Trend-following in medium-term trades (weeks to months)

Parameters:

short_period (default: 9) - Fast EMA

long_period (default: 21) - Slow EMA

2. RSI Strategy

File: 

backend/strategies/rsi_strategy.py

Description: Relative Strength Index measures overbought/oversold conditions.

Logic:

Buy Signal: RSI crosses above 30 (oversold recovery)

Sell Signal: RSI crosses below 70 (overbought cooling)

Confidence: Based on RSI momentum and trend strength

Use Case: Mean-reversion trading in ranging markets

Parameters:

period (default: 14) - RSI calculation period

overbought (default: 70) - Upper threshold

oversold (default: 30) - Lower threshold

3. MACD Strategy

File: 

backend/strategies/macd_strategy.py

Description: Moving Average Convergence Divergence tracks momentum and trend.

Logic:

Buy Signal: MACD line crosses above signal line

Sell Signal: MACD line crosses below signal line

Histogram: Shows momentum strength

Use Case: Momentum trading and trend confirmation

Parameters:

fast_period (default: 12)

slow_period (default: 26)

signal_period (default: 9)

4. Bollinger Scalping

File: 

backend/strategies/bollinger_scalping.py

Description: Mean reversion using Bollinger Bands.

Logic:

Buy Signal: Price touches lower band (oversold)

Sell Signal: Price touches upper band (overbought)

Bands: Price ± 2 standard deviations from 20-day SMA

Use Case: Short-term scalping in volatile markets

Parameters:

period (default: 20) - Moving average period

std_dev (default: 2) - Standard deviation multiplier

5. SuperTrend

File: 

backend/strategies/supertrend.py

Description: Trend-following indicator using ATR (Average True Range).

Logic:

Buy Signal: Price closes above SuperTrend line

Sell Signal: Price closes below SuperTrend line

Adaptive: Adjusts to volatility via ATR

Use Case: Strong trending markets, ride major moves

Parameters:

period (default: 10) - ATR period

multiplier (default: 3) - ATR multiplier

6. Ichimoku Cloud

File: 

backend/strategies/ichimoku_strategy.py

Description: Japanese multi-faceted indicator showing support, resistance, and momentum.

Logic:

Buy Signal: Price above cloud, conversion line > base line

Sell Signal: Price below cloud, conversion line < base line

5 Lines: Conversion, base, lagging, leading span A & B

Use Case: Comprehensive trend analysis, multiple timeframes

Parameters:

tenkan (default: 9) - Conversion line

kijun (default: 26) - Base line

senkou (default: 52) - Leading span

7. ADX DMI

File: 

backend/strategies/adx_dmi_strategy.py

Description: Average Directional Index measures trend strength.

Logic:

Buy Signal: ADX > 25, +DI > -DI (strong uptrend)

Sell Signal: ADX > 25, -DI > +DI (strong downtrend)

Strength: ADX value (25+ = strong trend)

Use Case: Filter trades by trend strength, avoid choppy markets

Parameters:

period (default: 14) - ADX calculation period

threshold (default: 25) - Minimum trend strength

8. VWAP (Volume-Weighted Average Price)

File: 

backend/strategies/vwap_strategy.py

Description: Intraday trading using volume-weighted average.

Logic:

Buy Signal: Price crosses above VWAP (bullish)

Sell Signal: Price crosses below VWAP (bearish)

Institutional: Shows average price paid by institutions

Use Case: Intraday and short-term trading

Parameters:

Uses daily volume-weighted calculations

9. Breakout Strategy

File: 

backend/strategies/breakout_strategy.py

Description: Identifies price breakouts from consolidation ranges.

Logic:

Buy Signal: Price breaks above resistance with high volume

Sell Signal: Price breaks below support with high volume

Volume Confirmation: Must exceed average by 1.5x

Use Case: Capture explosive moves after consolidation

Parameters:

period (default: 20) - Range calculation period

volume_multiplier (default: 1.5) - Volume confirmation

10. ML LSTM Strategy

File: 

backend/strategies/ml_lstm_strategy.py

Description: Hybrid strategy using LSTM neural network for predictions.

Logic:

Buy Signal: LSTM predicts upward trend with >60% confidence

Sell Signal: LSTM predicts downward trend with >60% confidence

AI-Powered: Learns from historical patterns

Use Case: Combine machine learning with technical analysis

Parameters:

lookback (default: 60) - Days of historical data

confidence_threshold (default: 0.6) - Minimum prediction confidence

Machine Learning Models

-Regression Models (Price Prediction)

1. LSTM (Long Short-Term Memory)

File: 

backend/models/lstm_model.py

Description: Deep learning recurrent neural network for time series.

Architecture:

2 LSTM layers (128, 64 units)

Dropout layers (0.2) to prevent overfitting

Dense output layer

Training:

Lookback: 60 days

Features: Close price

Loss: MSE (Mean Squared Error)

Accuracy: ~85-90% on historical data
Use Case: Short-term (1-7 day) price predictions

Description: Enhanced BiLSTM with OHLCV features for 7-day forecast.

Architecture:

Bidirectional LSTM (256, 128, 64 units)

Technical indicators (RSI, MACD, Bollinger Bands)

Dual output: Price + Trend classification

Features (40+ indicators):

Moving averages (SMA, EMA)

Volatility (ATR, Bollinger Bands)

Accuracy: 60-70% trend classification
Use Case: Weekly forecasting with trend direction
